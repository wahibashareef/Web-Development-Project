To-Do List Web Application

Project Overview:
The To-Do Web App allows users to efficiently manage their daily tasks. Users can add, new tasks, edit task dtails, mark them as 
completed, delete individual tasks, and clear all tasks. The app provides an intuitive interface for task management, helping 
users stay organized & productive.

Features:
- Add Task: Easily add new tasks to the list.
- Edit Task: User can modify the existing task.
- Delete Task: Remove a specific task from the list.
- Mark as completed: Check off tasks when completed.
- Clear all Tasks: Remove all the tasks from the list at once.

Technologie used:
- HTML: Structure the web page.
- CSS: Styles the application with a clean and responsive design.
- JavaScript: Handles task management, including adding, editing, deleting, and marking tasks as completed.

Project Structure:
- index.html: Main page of the application.
- styles.css: stylesheet for the application, ensuring a user-friendly layout.
- app.js: JavaScript file that manages task functionalities such as adding, editing, deleting, and marking tasks as completed.

Usage Instructions:
1. Adding a task: Enter a task in the input field and click add '+' button.
2. Editing a task: Click the edit icon next to the task, then a field is displayed for editing. Modify the text then click 'ok'.
3. Deleting a task: Click the delete icon next to the task to remove that task task from the list.
4. Marking as completed: Check the checkbox near the task to mark it as completed.
5. Clearing all tasks: Click 'Clear All' button to remove all the task from the list at once.

Future enhancements:
Include user authentication, task prioritization, and deadline reminders.
